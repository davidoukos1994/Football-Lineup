# Football Coach v8.1.8

## Διόρθωση
- Το popup παίκτη γράφει πλέον «Επιλογή από ρόστερ».
- Το dropdown ανοίγει όλο το ρόστερ, όχι μόνο την αποστολή.
- Αν ο παίκτης έχει δηλωμένη θέση, εμφανίζεται πρώτα στους προτεινόμενους για τη συγκεκριμένη θέση.
- Μπορείς πάντα να επιλέξεις οποιονδήποτε παίκτη από όλο το ρόστερ.
