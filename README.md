# Football Coach v8.1 Share Transfer

Βασισμένο στο v8.1.

## Νέα
- Κουμπί «Αποστολή / Μεταφορά δεδομένων»
- Δημιουργία αρχείου backup για αποστολή σε άλλη συσκευή
- Κοινή χρήση μέσω κινητού, όπου υποστηρίζεται:
  - Mail
  - Messenger
  - Viber
  - AirDrop
  - αρχεία
- Εναλλακτικά: Λήψη αρχείου backup
- Restore από αρχείο backup σε άλλη συσκευή

## Χρήση
1. Στην παλιά συσκευή πάτα «Αποστολή / Μεταφορά δεδομένων».
2. Πάτα «Κοινή χρήση αρχείου» ή «Λήψη αρχείου backup».
3. Στείλε το αρχείο στον εαυτό σου.
4. Άνοιξε την εφαρμογή στη νέα συσκευή.
5. Πάτα «Restore από αρχείο».
6. Διάλεξε το backup.
