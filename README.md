# Football Coach v8.1.6

## Διόρθωση
- Το «Διάλεξε παίκτη» στο popup παίκτη γεμίζει πλέον από το ρόστερ.
- Προτείνονται πρώτα οι παίκτες που έχουν δηλωμένη τη συγκεκριμένη θέση.
- Μπορείς πάντα να επιλέξεις οποιονδήποτε παίκτη από το ρόστερ.
- Όταν επιλέγεις παίκτη, περνάει αυτόματα όνομα και αριθμός.
- Δεν επηρεάστηκαν ρόστερ, παρουσιολόγιο, dashboard, αποθήκευση ή μεταφορά δεδομένων.
