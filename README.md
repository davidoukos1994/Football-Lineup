# Football Coach v7.2

Βασισμένο στο λειτουργικό v7.1.2.

## Νέα
- Πατάς πάνω σε παίκτη στο γήπεδο και ανοίγει popup επεξεργασίας
- Αλλαγή ονόματος από το popup
- Αλλαγή αριθμού από το popup
- Γκολ
- Κίτρινες / κόκκινες κάρτες
- Λεπτά συμμετοχής
- Βαθμολογία 1-10
- Εμφάνιση γκολ πάνω στο κουτί του παίκτη
- Εμφάνιση κάρτας πάνω στο κουτί του παίκτη
- Σταθερό μέγεθος κουτιών παικτών

## Διατηρήθηκαν
- Drag & drop
- Γήπεδο
- Πάγκος
- Σκορ
- Χρονόμετρο
- Backup / Restore
- Διαχείριση ιστορικού
