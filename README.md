# Football Lineup Builder Pro v4

## Νέα έκδοση

- Αριθμός φανέλας σε κάθε παίκτη.
- Μετακίνηση παικτών με drag & drop.
- Σημειώσεις αριστερά από το γήπεδο.
- Αυτόματη αποθήκευση στον browser.
- Επιλογή χρώματος για τα κουτάκια των παικτών.
- Σχεδίαση τακτικής με βέλη πάνω στο γήπεδο.
- Εξαγωγή σε PNG και PDF.
- Κουτάκια παικτών που μεγαλώνουν για μεγάλα ονόματα.
