# Football Lineup Builder

Μικρή web εφαρμογή για δημιουργία 11άδας ποδοσφαίρου.

## Δυνατότητες

- Γήπεδο ποδοσφαίρου 11x11
- Ονόματα παικτών σε κάθε θέση
- Αλλαγή συστήματος
- Αναπληρωματικοί
- Αλλαγή ονόματος ομάδας
- Upload σήματος ομάδας
- Εξαγωγή εικόνας PNG

## Πώς το ανεβάζεις στο GitHub

1. Δημιούργησε νέο repository στο GitHub.
2. Ανέβασε τα αρχεία `index.html`, `style.css`, `script.js`.
3. Πήγαινε Settings → Pages.
4. Στο Branch διάλεξε `main` και `/root`.
5. Πάτησε Save.

Μετά από λίγο θα έχεις link για την εφαρμογή σου μέσω GitHub Pages.
