# Football Coach v7.5.2

Διόρθωση αποθήκευσης σήματος και χειροκίνητη αποθήκευση.

## Νέα
- Το σήμα ομάδας μικραίνει αυτόματα πριν αποθηκευτεί.
- Κουμπί «Αποθήκευση τώρα».
- Καλύτερη υποστήριξη για κινητό/Safari.
- Μειώνεται ο κίνδυνος να γεμίσει το Local Storage.

## Χρήση
Αν ανεβάσεις μεγάλο σήμα, η εφαρμογή το μετατρέπει αυτόματα σε μικρότερο αρχείο ώστε να μπορεί να αποθηκευτεί στο κινητό.
