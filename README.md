# Football Coach v8.1.5

Διόρθωση αλφαβητικής σειράς και παρουσιολογίου.

## Διορθώσεις
- Όλοι οι παίκτες του ρόστερ εμφανίζονται αλφαβητικά.
- Το παρουσιολόγιο γεμίζει αυτόματα από το ρόστερ.
- Στο παρουσιολόγιο εμφανίζονται όλοι οι πραγματικοί παίκτες.
- Καθαρίζονται οι παίκτες τύπου «Παίκτης 1», «Παίκτης 2» από παντού.
- Οι καρτέλες παικτών συγχρονίζονται με το ρόστερ.
- Το Dashboard διαβάζει τα καθαρά δεδομένα.
