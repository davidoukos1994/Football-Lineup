# Football Coach v8.0 IndexedDB

Πρώτη έκδοση με νέα βάση αποθήκευσης.

## Τι αλλάζει
- Η εφαρμογή συνεχίζει να ανεβαίνει κανονικά στο GitHub Pages.
- Τα δεδομένα δεν αποθηκεύονται πλέον κυρίως στο LocalStorage.
- Προστέθηκε IndexedDB storage layer.
- Μεγάλα δεδομένα όπως σήμα, ρόστερ, ιστορικό, παρουσιολόγιο και αγωνιστικές μπορούν να αποθηκεύονται με πολύ περισσότερο χώρο.
- Η πρώτη φόρτωση μεταφέρει αυτόματα τα παλιά δεδομένα από LocalStorage σε IndexedDB.

## Τι δεν αλλάζει
- Ίδιο link στο GitHub Pages.
- Ίδια χρήση.
- Ίδιο upload αρχείων.
- Ίδιο layout.
- Ίδιες λειτουργίες v7.5.2.
