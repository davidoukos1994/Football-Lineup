# Football Lineup Builder v2

## Νέα v2
- Τα κουτάκια των παικτών μεγαλώνουν αυτόματα για μεγάλα ονόματα.
- Προστέθηκε πλαίσιο σημειώσεων αριστερά από το γήπεδο.
- Οι παίκτες μετακινούνται με drag & drop μέσα στο γήπεδο.
- Διπλό κλικ πάνω σε παίκτη για γρήγορη αλλαγή ονόματος.
- Η εξαγωγή εικόνας περιλαμβάνει γήπεδο, σημειώσεις, όνομα ομάδας, σήμα και αναπληρωματικούς.
